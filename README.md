# United Kingdom Top 50 Playlist Market Structure, Artist Diversity & Content Localization Analysis

## Project Overview

This project analyzes the structural composition of the United Kingdom Top 50 music playlist to uncover artist dominance, collaboration behavior, content localization patterns, and release strategy effectiveness. The dashboard provides actionable market intelligence to support UK-focused strategic decisions for Atlantic Recording Corporation.

---

## Business Problem

Despite access to daily UK Top 50 playlist data, stakeholders lack clarity on:

- Distribution of artist dominance  
- Collaboration impact on chart performance  
- Cultural sensitivity toward explicit content  
- Effectiveness of single vs album release strategies  

This project bridges that gap by transforming playlist data into decision-ready insights.

---

## Core Features

- Market Concentration KPIs  
- Artist Dominance Leaderboard
